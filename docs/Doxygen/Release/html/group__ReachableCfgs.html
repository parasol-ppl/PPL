Reachable Distance/Volumn Space (RD/V-space).

Various configuration types which are related to articulated linkages and closed chain robotic systems. Reachable Distance is a theoretical foundation for sampling configurations from Reachable Distance Space (RD-space). These configuration classes represent this concept.
