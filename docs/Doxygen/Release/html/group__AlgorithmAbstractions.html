Detailed Description

The algorithm abstractions in PMPL are:

  • Distance Metrics - Compute distances between two configurations
  • Validity Checkers - Determine if a configuration is in or
  • Samplers - Generate configurations in (usually )
  • Neighborhood Finders - Determine nearest neighbors within a roadmap
  • Local Planners - Validate simple, often deterministic paths, between configurations
  • Extenders - Step configurations toward a direction until an end condition is met, e.g., a distance is reached
  • Connectors - Generate edges of a roadmap, usually a combination of a neighborhood finder and a local planner
  • Metrics - Determine a value/property from a roadmap
  • Map Evaluators - Determine is a roadmap passes an evaluation test, e.g., can solve an example query
  • Path Modifiers - Smooth/modify an input path to produce a new valid path.
  • Region Decomposition - Region framework to split space for divide and conquer methods
  • Motion Planning Strategies - Loosely, construct a roadmap, but more generally are anything executable by PMPL, e.g., solve this problem or smooth this path
